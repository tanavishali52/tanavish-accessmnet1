# NexusAI: Agentic Platform Architecture

NexusAI is a centralized platform for discovering, comparing, and deploying cutting-edge AI models. It features a rich, interactive chat interface (Chat Hub), a comprehensive model marketplace, and an automated Agent Builder.

## 📁 Repository Structure

The project is organized into two primary workspaces:
- **`nexus-ai-hub/`**: Frontend workspace (Next.js 14, Redux Toolkit, TailWind CSS, Framer Motion).
- **`nexus-ai-backend/`**: Backend API (NestJS, MongoDB/Mongoose, Standardized REST API).

## 🛠 Technology Stack

### Frontend Hub
- **Framework**: Next.js (App Router).
- **State**: Redux Toolkit for messages, onboarding, and catalog data.
- **Styling**: TailWind CSS v4 + HSL-curated CSS variables.
- **Animations**: Framer Motion (Transitions, Micro-interactions).
- **UI System**: `Syne` (Headings), `Instrument Sans` (Body), `React Icons/Fi`.

### Nest Backend
- **Framework**: NestJS (Modular Architecture).
- **Database**: MongoDB via Mongoose ODM.
- **Response Format**: Standardized `{ success: boolean, message: string, data?: any }`.
- **Validation**: Global `HttpExceptionFilter` and `ResponseInterceptor`.

## 📜 Development Rules

1. **[Code Style](file:///D:/nexes.ai/.claude/rules/code-style.md)**: PascalCase for components, camelCase for hooks, and strictly typed TypeScript.
2. **[Security](file:///D:/nexes.ai/.claude/rules/security.md)**: No hardcoded secrets, mandatory auth guarding, and XSS prevention.
3. **[Error Handling](file:///D:/nexes.ai/.claude/rules/error-handling.md)**: Normalized response patterns and graceful UI failure states.

## 🚀 Specialized Skills

- **[Responsive Design](file:///D:/nexes.ai/.claude/skills/responsive-design.md)**: Mobile-first layouts using HSL variables and Tailwind.
- **[API Integration](file:///D:/nexes.ai/.claude/skills/api-integration.md)**: Standardized `request<T>` utility and Redux status patterns.
- **[PR Review](file:///D:/nexes.ai/.claude/skills/pr-review.md)**: Final quality checklist before merging features.
- **[Routing & Auth](file:///D:/nexes.ai/.claude/skills/frontend-routing-auth.md)**: Next.js App Router navigation and session hydration.

## 📜 Core Standards

1. **Safety First**: Use TypeScript's strict-type definitions for all API responses and component props.
2. **Standardized Communication**: All frontend-to-backend calls must use the internal `request` utility.
3. **Responsive first**: UI components must behave perfectly on mobile/tablet before desktop optimizations.
4. **Design Excellence**: Adhere strictly to the brand typography and HSL color palette.
